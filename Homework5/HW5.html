Multiplication Table

Below, please enter your desired upper-bound and lower-bound numbers for each axis of the multiplication table that you want to display. The value of the X upper-bound must exceed the value of the X lower-bound and the value of the Y upper-bound must exceed the value of the Y lower-bound. All four fields must be filled in. Each number should be no lesser than -150 and no greater than 150. You may not enter a decimal or a number containing E or e. Failure to follow these instructions will result in an error message when the submit button is pressed.
